miilClient
=====
# 準備
```
$ git clone https://github.com/daiz713/miilClient.git
$ cd miilClient
$ sh setup.sh
```

# Androidアプリとして起動
MacにAndroidデバイスを接続して, `miilClient/`において下記のコマンドを実行.
```
$ python universe.py android
$ cordova run android
```

# Chromeアプリとして起動
`miilClient/`において下記のコマンドを実行.
```
$ python universe.py chromeapp
$ chrome run www
```
